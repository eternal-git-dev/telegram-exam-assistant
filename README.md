# Telegram-бот «Помощник по подготовке к экзаменам»

Асинхронный Telegram-бот для помощи пользователям в подготовке к экзаменам.  
Проект демонстрирует полный backend-workflow: сбор заявок, управление справочными данными,
модерацию пользователей и административную панель.

> ⚠️ **Дисклеймер**  
> Данный проект является демонстрационным.  
> Логика работы моделирует процесс подготовки к экзаменам и консультаций
> в образовательном контексте.

---

## Возможности

### Для пользователей
- Пошаговый процесс создания заявки на подготовку к экзамену
- Выбор:
  - образовательной платформы / университета
  - предмета
  - формата подготовки (консультация, разбор тем и т.д.)
  - желаемого срока или дедлайна
- Просмотр своих заявок и их текущего статуса
- Удобный Telegram-интерфейс с FSM-логикой

### Для администратора
- Модерация заявок пользователей
- Управление справочными данными:
  - университеты / платформы
  - предметы
  - форматы подготовки
- Управление пользователями (бан / разбан)
- Уведомления о новых заявках

### Технические возможности
- Полностью асинхронная архитектура
- FSM (Finite State Machine) для диалоговых сценариев
- Redis-кэширование с TTL
- Middleware для контроля доступа (администратор, бан)
- Чёткое разделение ответственности между слоями
